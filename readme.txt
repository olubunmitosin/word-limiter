A wordpress plugin that helps restrict character inputs for supported editors.

== Description ==
A wordpress plugin that helps restrict character inputs for supported editors. This is important when you want to limit the number of character a user can enter in posts box, comment and user admin editors.

== Installation ==
Download plugin from wordpress directory

== Upgrade Notice ==
This plugin is updated to enable capabilities with latest release of Core Wordpress
